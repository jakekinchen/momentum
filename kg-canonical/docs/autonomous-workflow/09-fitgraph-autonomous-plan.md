# 09 FitGraph Autonomous Plan

## M0 - KG Module Walking Skeleton

Create a minimal Python project with typed modules, seed file placeholders,
validation entrypoints, and tests that prove the package imports and exposes
graph health.

## M1 - Resolver And Seed Graph

Implement local term resolution, anatomy closure, exercise-family closure,
equipment parsing, and unresolved concept behavior.

## M2 - Safety Engine And Decision Receipts

Implement deterministic hard blocks, soft penalties, all-reason collection,
severity lattice selection, and constraint fingerprints.

## M3 - Alternatives And Workout Candidate API

Select alternatives from the safe pool and expose the workout-candidate contract.

## M4 - Member Context And Copilot Fact Cards

Implement source-backed member facts, direct graph queries, fact cards, and
Copilot retrieval routing.

## M5 - Ontology Sidecar And Validation

Pin ontology lock metadata, add SKOS-style mappings, add PROV-shaped receipts,
and add optional RDF/Turtle export plus validation checks.
