Current assessment

  • Geometry and timing port cleanly because the SwiftUI pose is formula-driven, not based on sampled video.
  • High pixel similarity is blocked by browser and SwiftUI differences in SF Symbols, system colors, text antialiasing, Canvas strokes, and shadow blur.
  • For a 95% loop, capture fixed native frames and compare against deterministic browser frames at the same size, scale factor, phase, font stack, and color profile.
