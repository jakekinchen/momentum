# FitGraph Autonomous Workflow

This directory defines the repo-local Codex Executor / Reviewer / Manager
workflow for FitGraph.

Start here:

- `../kg-module-prd.md` for product requirements.
- `../../GOAL.md` for current mission and active slice.
- `../../executor-reviewer-pair-programming.md` for role boundaries.
- `01-operating-model.md` through `08-scaffold-adoption-matrix.md` for reusable
  workflow rules.
- `09-fitgraph-autonomous-plan.md` for milestone sequencing.

The workflow is intentionally evidence-heavy. A slice is not complete until the
executor leaves a session log, validation output, and a scoped commit, and the
reviewer leaves an exact decision.
